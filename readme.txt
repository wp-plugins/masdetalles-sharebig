== Description ==

The MasDetalles ShareBig plugin to help people share, bookmark, and email your posts and pages using Masdetalles share.
New: MasDetalles ShareBig.

MasDetalles ShareBig le permite compartir sus enlaces en el marcador social Masdetalles.do desde su blog, email, tienda virtual, y cualquier pagina web.

== Installation ==

1. Upload the MasDetalles-ShareBig directory (including all files within) to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress

Spanish:
1. Sube el directorio MasDetallesShareBig (incluyendo todos los archivos) al directorio /wp-content/plugins/
2. Activa el plugin en el menu "Plugins" de WordPress.
